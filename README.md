# SecureBank-Pro
SecureBank Pro is a modular, multi-user, file-based banking management system implemented in C
## Project Overview
SecureBank Pro is a modular, multi-user, file-based banking management system implemented in C. The system simulates core banking operations such as account creation, authentication, deposit, withdrawal, fund transfer, and transaction logging with persistent data storage.

## Problem Statement
To design and implement a secure, console-based banking system using structured programming principles and file handling to manage multiple user accounts efficiently.

## Key Features
- Account creation and deletion
- Secure login with PIN authentication
- 3-attempt blocking mechanism
- Deposit and withdrawal with validation
- Fund transfer between accounts
- Transaction history logging
- Admin panel for account management
- Persistent data storage using binary files

## Technologies Used
- C Programming Language
- Structured Programming
- File Handling (Binary Files)
- Modular Design

## Libraries
- stdio.h
- stdlib.h
- string.h
- time.h

## System Architecture
The system follows a layered structure:
1. Presentation Layer (Console UI)
2. Business Logic Layer
3. Data Persistence Layer

## Module Division
- Authentication & Security
- Account Management
- File Storage System
- Transaction Engine
- Transfer & Logging
- Admin Panel & Documentation
