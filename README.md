SecureBank
Overview

SecureBank is a console-based banking system developed in C. It allows users to create accounts and perform basic banking operations using file storage.

Features
Create new account
Login with account number and PIN
Deposit money
Withdraw money
Transfer money
Check balance
Update account details
Admin panel to view all accounts
Technologies Used
C Programming
File Handling
Basic Encryption (XOR)
How to Run

Compile:
gcc secure_bank.c -o secure_bank

Run:
./secure_bank

Data Storage

All data is stored in a file (accounts.txt) with basic encryption.
